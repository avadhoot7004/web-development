Javascript objects

In JavaScript, an object is a collection of properties, where each property consists of a key (also known as a property name) and a corresponding value. Objects can store multiple values in a single variable and are a fundamental part of JavaScript.

person = { firstName: "Morgan", lastName: "Jones", age: 45 }; person.fullName = function() { return this.firstName + " " + this.lastName; };

String methods

A JavaScript string is zero or more characters written inside quotes.

There are many string methods as follows:

  • .length:
     The length property returns the length of a string. Consider string "ABCDEFGHIJKLMNOPQRSTUVWXYZ"  

  • substring():
     Returns the characters in a string between two indexes into the string. Consider the string "Hello World".

  • toLocaleLowerCase():
     The characters within a string are converted to lower case while respecting the current locale. Consider the string "HELLO WORLD".

  • toLocaleUpperCase():
     The characters within a string are converted to upper case while respecting the current locale. Consider the string "hello world".

  • toLowerCase():
     Returns the calling string value converted to lower case. Consider the string "HELLO WORLD".

  • toString():
     Returns a string representing the specified object. Consider the number 123.

  • toUpperCase():
     Returns the calling string value converted to uppercase. Consider the string "hello world".

  • valueOf():
     Returns the primitive value of the specified object. Consider the string "Hello".
